## 🛠️ Troubleshooting Common Issues

**❌ Bunni Executor won't attach to Roblox**
This usually happens when the tool is not running with administrator privileges or when the Roblox client has not fully loaded yet.
**Solution:** Right-click `BunniExecutor.exe`, select **Run as Administrator**, and wait until the Roblox experience is fully loaded before attempting to attach.

**❌ Antivirus blocks or deletes the file**
Most antivirus programs flag script executors as potentially unwanted software due to the nature of code injection.
**Solution:** Add the entire `C:/BunniExecutor/` folder to your antivirus exclusion list. This is a false positive — **Is Bunni Executor Safe** is covered in detail in the community FAQ and independent reviews.

**❌ Scripts fail with "Execution Error" in console**
The script may be outdated, incompatible, or contain syntax errors that prevent it from running properly.
**Solution:** Try a different script from the built-in hub first. If the issue persists, restart Bunni Executor and re-attach to a fresh Roblox session.

**❌ Android `.apk` installation fails**
Newer Android versions restrict installations from unknown sources by default.
**Solution:** Navigate to `Settings > Apps > Special Access > Install Unknown Apps` and grant permission to your file manager or browser. Then retry the Bunni Executor `.apk` install.

**❌ UI appears blurry or oversized**
Display scaling above 100% on Windows can cause visual glitches in the executor window.
**Solution:** Right-click `BunniExecutor.exe`, go to `Properties > Compatibility > Change high DPI settings`, and enable **Override high DPI scaling behavior**.
